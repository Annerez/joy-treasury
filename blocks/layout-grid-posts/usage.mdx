## How to use

The `LayoutGridPosts` accepts 3 children which gives priority to the first item. The children must be the same html tag.

```jsx
import LayoutGridPosts from "./path/to/LayoutGridPosts";

function App() {
  return (
    <LayoutGridPosts>
      <div />
      <div />
      <div />
    </LayoutGridPosts>
  );
}
```
