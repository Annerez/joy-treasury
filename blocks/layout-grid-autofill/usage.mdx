## How to use

The `LayoutGridAutofill` is a CSS Grid container that automatically autofills the children.

```jsx
import LayoutGridAutofill from "./path/to/LayoutGridAutofill";

function App() {
  return (
    <LayoutGridAutofill>
      <div />
      <div />
      <div />
      ...
    </LayoutGridAutofill>
  );
}
```

## Props

### `size`

The grid item size. Default is `md`.

```ts
size?: 'sm' | 'md' | 'lg'
```
