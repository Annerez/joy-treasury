## How to use

The `LayoutGridPosts2` accepts 5 children which gives priority to the first item. The children must be the same html tag.

```jsx
import LayoutGridPosts2 from "./path/to/LayoutGridPosts2";

function App() {
  return (
    <LayoutGridPosts2>
      <div />
      <div />
      <div />
      <div />
      <div />
    </LayoutGridPosts2>
  );
}
```
